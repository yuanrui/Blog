### 数据类型



| 数据类型 | 存储大小 | 取值范围 | 默认值 | C#类型 |      |
| -------- | -------- | -------- | ------ | ------ | ---- |
| byte                      | 1字节 |          | 0         |        |      |
| short                     | 2字节 |          | 0         |        |      |
| int                       | 4字节 |          | 0         |        |      |
| long                      | 8字节 |          | 0L        |        |      |
| float                     | 4字节 |          | 0.0f      |        |      |
| double                    | 8字节 |          | 0.0d      |        |      |
| char                      |          |          | '\u0000'  |        |      |
| String (or any object)    |          |          | null      |        |      |
| boolean                   |          |          | false     |        |      |
